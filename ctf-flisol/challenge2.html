🏰 Descripción

La fortaleza de Doom tiene un sistema de login vulnerable. Tu misión es bypassear la autenticación usando SQL Injection.

⚠️ Este formulario es vulnerable a SQL Injection básica. Solo para aprendizaje.

🔐 Login de Latveria

💉 Vulnerabilidad

El formulario es vulnerable a SQL Injection. La query backend es algo como:

SELECT * FROM users WHERE username = '' + input + '' AND password = '...'

💡 Pistas

Prueba con este username:

admin' OR '1'='1' --

El ' cierra el string, OR '1'='1' hace la condición siempre verdadera, y -- comenta el resto de la query.

La query se convierte en:

SELECT * FROM users WHERE username = 'admin' OR '1'='1' --' AND password = '...'

El OR '1'='1' siempre es TRUE, entonces devuelve todos los users.

El -- comenta todo lo que sigue, incluyendo el AND password = '...'

Así que con el payload correcto, el password NO importa. Puedes poner lo que quieras.

⚡ Solver Rápido

1. En Username, ingresa: admin' OR '1'='1' --

2. En Password, ingresa cualquier cosa: xxx

3. Haz click en INICIAR SESIÓN

La flag aparecerá si el bypass es exitoso.
