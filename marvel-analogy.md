## 🔗 RECURSOS PARA EMPEZAR

### Como Tony empezó en su garaje:

1. **TryHackMe** (tryhackme.com) - Labs gratuitos, nivel principiante
2. **PortSwigger Academy** (portswigger.net/web-security) - 300+ labs
3. **HackTheBox** (hackthebox.com) - Máquinas para practicar
4. **DVWA** (Docker: vulnerables/web-dvwa) - Laboratorio local

### Como Dr. Doom se prepara:

1. Aprende HTTP y cómo funciona la web
2. Practica en DVWA/Juice Shop
3. Aprende Burp Suite
4. Únete a Bugcrowd VDP (sin bounty, solo práctica)
5. Empieza en Intigriti (pagos reales)
